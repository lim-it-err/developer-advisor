# 도메인 후보군 카탈로그

> "세상을 거대한 디지털 구조로 본다" — 미션 도메인 후보의 전체 풀.
> 선택 기준: ① 공부해야만 짤 수 있는 규칙이 있는가 ② 브리핑이 상식으로서 재미있는가 ③ 스테이지 훈련 주제를 자연스럽게 강제하는가.
> 상태: 💚 미션 존재 · 🎯 백로그 확정 · 💡 후보

## 🎼 음악 · 합창단 🎯
- **인트로 (확정)**: 어느 개발자가 카페에서 코드를 짜는데, 옆 테이블 합창단원들이 파트 연습 일정으로 다투고 있었다. 소프라노 부족, 베이스 과잉, 연습실 하나. 개발자는 이어폰을 뺐다 — "이건 스케줄링 문제잖아."
- 파트 배정기 (SATB 음역·균형 — 제약 충족), 조옮김 엔진 (조표·임시표 규칙), 연습 일정 스케줄러
- 평균율 vs 순정률 — 완벽한 화음의 수학적 불가능과 오차 균등 분배. 부동소수점 반올림의 조상 (브리핑 최상급 소재)
- 스트리밍 로열티 분배 (저작권료 정산 — 금융 크로스), 아이돌 티켓팅 (동시성)

## 🚄 철도 확장 (사용자 최애)

- 🎯 시베리아 횡단철도 — 전 구간 모스크바 시간 운행 역사 (시간대 도메인의 성지)
- 💡 스위스 시계식 다이어그램(Taktfahrplan) — 전국 매시 정각 환승 스케줄링
- 💡 인도 IRCTC — 초당 수만 예매, 대기열·동시성
- 💡 1964 신칸센 — 도쿄올림픽 D-9 개통, 데드라인 드리븐 개발의 원형
- 💡 수도권 전철 개통 1974 — 국철·지하철 직결 = 두 시스템의 인터페이스 협상
- 영화 크로스: 설국열차, 부산행

## 🎬 영화 오마주
- 🎯 **설국열차** — 열차 칸 = 접근 권한 계층. 꼬리칸의 엔진칸 API 호출 (인증/인가)
- 🎯 **마션** — 화성 감자 자원 계산기 + 16진수 아스키 통신 (제약 하 생존 로직)
- 💡 머니볼 — 출루율 선수 평가, 스카우팅 레거시 교체
- 💡 히든 피겨스 — 수기 계산 → IBM 전환, 레거시 마이그레이션의 원형 (S4 각)
- 💡 아폴로 13 — 주어진 부품만으로 해결 (제공 코드 제약 미션)
- 💡 어벤져스 — 타노스 스냅 = 노드 절반 장애에도 도는 설계 (S5)
- 💡 쥬라기 공원 — 사육장 접근 제어 + 한 명에게 몰린 권한("데니스 네드리 문제")
- 💡 인터스텔라 — 시간 팽창 타임스탬프 정합성
- 💡 백 투 더 퓨처 — 날짜 연산 버그 시리즈
